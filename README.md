
# [vidimaze.quest](https://vidimaze.quest/)

# **About**
>This web app was made for anyone who wants to find new youtube media without recommendations influencing search results. A place where you can maybe find new things and even a ranking page so you can see what others have found.

- Backend:
<code>Java</code>
<code>Spring</code>

<code>PostgreSQL</code>
- API:
<code>YouTube Data API v3</code>
